# 🍲 Annapurna Edits

A modern and interactive recipe blogging website built using **HTML, CSS, and JavaScript**.

Annapurna Edits allows users to explore delicious recipes, filter them by categories or seasons, search recipes instantly, and interact through likes and comments — all within a clean and elegant food-blog UI.



## ✨ Features

- 🔍 Real-time recipe search
- 🍽️ Category-based filtering
- 🌦️ Seasonal recipe filters
- ❤️ Like system
- 💬 Comment section
- 🖼️ Recipe popup modal
- ➕ Add/Edit/Delete recipes
- 📸 Image upload preview
- 🔐 Login & Logout interface
- 🎨 Responsive and modern UI



## 🛠️ Tech Stack
